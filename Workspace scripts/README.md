# Maever_CSS/MAFIA Video Website

This is a static HTML website that autoplays a selected MP4 video.

## Features
- Autoplay video (muted to comply with browser policies)
- Self-hosted on GitHub Pages (free hosting)
- Runs 24/7 without cost

## Deployment to GitHub Pages
1. Create a new repository on GitHub.
2. Push this project to the repository.
3. Go to repository Settings > Pages.
4. Set source to "Deploy from a branch" and select main branch.
5. The site will be available at `https://yourusername.github.io/repositoryname/`

## Local Testing
Open `index.html` in your browser to test locally.
